# Gene Correlation Explorer

Interactive Shiny application for exploring gene–gene correlations.

## Requirements
- R ≥ 4.2
- RStudio (recommended)

## Setup
1. Download this folder
2. Download the required data files into the `data/` directory

https://cornell.box.com/s/xex2urj4ot6zmworcaq63vn4tbnp5qw9

3. Double-click `GeneCorrelationExplorer.Rproj`
4. Click **Run App**

The GUI will open automatically in your browser.
